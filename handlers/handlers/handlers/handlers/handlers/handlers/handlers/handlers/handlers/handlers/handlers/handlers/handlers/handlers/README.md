## 🚀 Deploy လုပ်နည်း

### Render မှာ Deploy လုပ်ခြင်း
1. GitHub Repository ကို ဖန်တီးပါ
2. Render.com မှာ အကောင့်ဖွင့်ပါ
3. New Web Service ကိုနှိပ်ပြီး GitHub Repo ကိုချိတ်ပါ
4. Environment Variables ထည့်ပါ:
   - `BOT_TOKEN` - သင့် Bot Token
   - `HF_TOKEN` - Hugging Face Token
   - `PORT` - 8080
5. Deploy လုပ်ပါ

### UptimeRobot နဲ့ 24/7 အိပ်မပျော်အောင်
1. UptimeRobot.com မှာ အကောင့်ဖွင့်ပါ
2. Monitor အသစ်လုပ်ပါ
3. URL: `https://your-bot-name.onrender.com`
4. Interval: 30 minutes
